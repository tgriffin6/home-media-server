# home-media-server
Automated Home Media Server in Docker Containers

# Container Stack
- Download Clients
    + qBittorrent (torrents)
    + SABnzbd (usenet)
- Servarr (media management)
    + Radarr (movies)
    + Sonarr (shows)
    + Prowlarr (indexers)
- Homepage (Dashboard)
- Plex (Media Playback aka the star of the show)
